* Akamai Delivery Property Onboard with Terraform

** Path Structure
#+begin_example
.
├── delivery                  -> Delivery onboard code
├── logic_module              -> Terraform module with logic rules
│   └── delivery_rules           -> Property manager logic rules
└── modules                   -> Terraform modules to configure Akamai platform
    ├── dns                      -> Akamai EdgeDNS
    ├── enrollment               -> Akamai CPS DV certificate
    ├── property                 -> Akamai Property manager
    └── validation               -> Akamai CPS certificate validation
#+end_example

** Use
- Init terraform inside "delivery" path
- Run other terraform commands inside "delivery" path.

** WorkFlow
1. Create cpcode based on property name.
2. Create enrollment for DV certificate.
3. Create edgehostnames enhanced-tls.
4. Create delivery property.
5. Push to Akamai staging and production.
